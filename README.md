# Google Professional Cloud Architect (PCA) Prep

This repository contains study materials, cheat sheets, and strategic whitepapers designed to assist in preparing for the Google Cloud Professional Cloud Architect certification exam.

## Contents

### Strategic Whitepapers & Guides
*   **[Modernizing Enterprise Security](modernize-enterprise-security.md)**: A comprehensive strategic framework for implementing identity-centric security using Google Cloud IAM. Covers Zero Trust, BeyondCorp, Workload Identity Federation, and governance best practices.
*   **[GCP Services Availability](gcp-services-availability.md)**: A reference guide covering availability concepts and service level agreements (SLAs) for key Google Cloud services.

### Cheat Sheets
*   **[IAM & Cloud Identity Cheat Sheet](cheatsheets/iam-cloud-identity-cheat-sheet.md)**: A "10-minute recollection" guide summarizing critical IAM concepts, resource hierarchy, policy inheritance, and identity federation patterns.

### Directories
*   `cheatsheets/`: Quick reference guides for last-minute review.
*   `pdf/`: Destination for PDF exports of the study materials.
*   `images/`: Diagrams and visual aids used in the documentation.

## Usage
These documents are written in Markdown and can be viewed directly in a code editor or rendered to PDF for offline study.
